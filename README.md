# 📝 Personal Notes App (React Edition)

Selamat datang di **Personal Notes App**! Proyek ini adalah aplikasi manajemen catatan pribadi berbasis web yang dibangun menggunakan **React** dan **Vite**. Proyek ini merupakan bagian dari perjalanan belajar saya di **DBS Foundation Coding Camp 2026**.

## 🚀 Fitur Utama
* **Dashboard Catatan:** Melihat semua daftar catatan aktif dengan tampilan yang bersih.
* **Tambah Catatan:** Form khusus untuk membuat catatan baru dengan mudah.
* **Detail Catatan:** Halaman khusus untuk membaca isi catatan secara lengkap berdasarkan ID.
* **Fitur Arsip:** Memindahkan catatan yang sudah selesai ke halaman arsip agar dashboard tetap rapi.
* **Halaman 404:** Navigasi cerdas jika pengguna mengakses link yang tidak tersedia.
* **Routing Dinamis:** Navigasi antar halaman yang cepat tanpa reload menggunakan `react-router-dom`.

## 🛠️ Teknologi yang Digunakan
* **Core:** React 18
* **Build Tool:** Vite (Super fast!)
* **Routing:** React Router DOM v7
* **Parsing:** html-react-parser (Untuk render konten HTML dalam catatan)
* **Fonts:** Inter (Google Fonts)

## 💻 Cara Menjalankan Proyek di Lokal
1. Clone repository ini.
2. Buka terminal dan jalankan `npm install` untuk mengunduh semua dependencies.
3. Jalankan `npm run dev` untuk memulai development server.
4. Buka `http://localhost:5173` di browser kamu.
